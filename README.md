

# SaberMais

### Equipe: Heitor Torres, Mariah Gabriella, Pedro Luiz, Ian Bomfim, Abner Gonçalvez

## Visão Geral

SaberMais é um aplicativo que lista importantes conteúdos e matérias do ensino brasileiro e os empacota de forma organizada e compreensiva de forma gratuita. O aplicativo organiza os conteúdos por matéria e também os classifica com nível de importância em relação ao Exame Nacional do Ensino Médio (ENEM). Além disso, a plataforma também oferece uma ordem predefinida para o aprendizado, implementado para evitar que os usuários se percam na pergunta de que conteúdo devem aprender primeiro, assim como replicar um pouco do ambiente escolar.

## Objetivo do App

Muitas pessoas possuem acesso à internet, porém não sabem por onde começar a estudar ou como organizar o aprendizado. O aplicativo tem como objetivo ajudar pessoas com acesso limitado à educação estudarem de forma gratuita, organizada e estruturada, oferecendo trilhas de aprendizado e recomendação de conteúdos educacionais disponíveis online. A plataforma irá orientar os usuários sobre o que estudar, em qual ordem estudar e onde encontrar conteúdos gratuitos na web, permitindo que mais pessoas tenham acesso ao conhecimento e possam desenvolver suas habilidades acadêmicas.
Dessa forma, o aplicativo contribui para democratizar o acesso à educação, ajudando estudantes que não têm oportunidades educacionais tradicionais.

## ODS Relacionado

ODS 4 – Educação de Qualidade

O aplicativo pretende auxiliar pessoas que não possuem acesso a escolas de qualidade, não possuem condições financeiras, ou 
materiais didáticos adequados por meio de uma estrutura de aprendizado guiada e conteúdos gratuitos disponíveis na internet, a plataforma permitirá que mais pessoas tenham a oportunidade de aprender. 
Assim, o projeto contribui para reduzir desigualdades educacionais e ampliar o acesso ao aprendizado.

## Diferencial

O aplicativo se diferencia das outras opções ao oferecer um caminho que o usuário pode seguir para aprender melhor. Ao invés de apenas fornecer os conteúdos, SaberMais atribui uma ordem a eles. Isso é feito para tornar a aprendizagem mais simples e replicar um pouco do sistema escolar.

## Requisitos Funcionais

O usuário pode selecionar uma matéria e visualizar os conteúdos presentes nela  
O sistema deve mostrar os tópicos em ordem de aprendizado  
O usuário pode marcar um tópico como concluído  
O sistema deve fornecer links gratuitos para cada tópico  
O sistema deve funcionar sem login

## Requisitos Não Funcionais

O aplicativo deve funcionar bem em dispositivos velhos e de baixa performance  
O aplicativo deve possuir uma fonte legível e acessível  
A navegação deve ser intuitiva   
Os links devem ser atualizados no caso de algum problema  
O sistema deve ser feito em Kotlin  
O sistema deve ser compatível com Android 8+  
O sistema salva os dados localmente
O sistema usará uma API externa
